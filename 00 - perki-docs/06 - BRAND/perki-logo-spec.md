# PERKI LOGO — SPEC & USAGE
### Status: 🔒 LOCKED (flat primary mark) · 2026-06-03

The flat vector mark is the official Perki logo and the workhorse across product, UI, and small sizes.

## Files
- `perki-icon.svg` — the coin mark (indigo squircle + gold P). App icon, favicon, avatar, social profile.
- `perki-logo-horizontal.svg` — full lockup (mark + "Perki" wordmark).

## The mark
- An indigo "coin" (rounded square) holding a gold **P** — a trusted vault surfacing the value (gold) inside.
- Construction: 512 grid · coin `rect 36,36 440×440 rx128` · P built from two `#E0A93B` strokes (`stroke-width 56`, round caps): stem `M211 150 V374`, bowl `M239 154 A62 62 0 0 1 239 278`.

## Colours (exact)
| Role | Hex |
|------|-----|
| Coin / brand | Indigo `#2B2A6E` |
| P / value accent | Gold `#E0A93B` |
| Surface | Eggshell `#F4F0E6` |
| Body text | Ink `#23202A` |

## Clear space & sizing
- Clear space ≥ 25% of the icon's width on all sides.
- Minimum icon size: 20px (favicon) — the mark still reads; verified in preview.

## Do
- Use the icon on eggshell, white, or photography with enough contrast.
- Use the all-indigo or all-gold monochrome version where colour is limited.

## Don't
- No concentric rings, gradients, or drop shadows on the flat mark (rings removed by decision).
- Don't recolour the P or coin outside the locked hex.
- Don't stretch, rotate, or add outlines to the wordmark.

## Wordmark
"Perki" — IBM Plex Sans, weight 600, in indigo `#2B2A6E`. (Headline pairing to be confirmed in Phase 2 / Pro Max.)
