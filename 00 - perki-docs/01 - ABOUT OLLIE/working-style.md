# WORKING STYLE

- Data first, narrative second.
- Start with the end-state visualisation, then work backwards.
- Ask clarifying questions early.
- Build shared artefacts to align teams.
- Prefer structured cadences over ad-hoc chaos.
- Calm under pressure; prioritises ruthlessly.
- Hands-on with data, systems, and prototypes.
- Uses AI as a force multiplier, not a novelty.
