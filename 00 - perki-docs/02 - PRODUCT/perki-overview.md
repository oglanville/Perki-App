# PERKI — PRODUCT OVERVIEW

Perki is a personal perks aggregator that consolidates all the benefits, discounts, and hidden 
features across a user’s subscriptions, memberships, and services.

The goal: 
Help users unlock value they’re already paying for but rarely use.

Current status:
- App built
- Email prototype built
- Supabase backend connected
- Static Excel perks being ingested
- Lidl + O2 perks pending
