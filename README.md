# Perki

All your membership perks in one place.

Perki aggregates membership perks from OVO Energy, Monzo, Revolut, and American Express into a single mobile-friendly app. Track which perks you've used, discover new ones, and never miss a benefit again.

## Local Development

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Deploy to Vercel

Push this repo to GitHub, then import it at [vercel.com/new](https://vercel.com/new). Vercel auto-detects Vite — no configuration needed.

## Tech Stack

- React 18
- Vite
- Deployed on Vercel
